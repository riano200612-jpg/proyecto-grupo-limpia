# Grupo Limpia - Limpieza Corporativa Premium en Cartagena

Sitio web profesional para Grupo Limpia, empresa especializada en servicios de limpieza corporativa, post-obra, hoteles y restaurantes en Cartagena de Indias.

## Características

✨ **Diseño Responsivo** - Optimizado para mobile, tablet y desktop
🎥 **Videos Profesionales** - Galería de estándares de calidad
📱 **WhatsApp Integrado** - Contacto directo con cotizaciones
🎯 **Formulario de Contacto** - Validación y envío automático
📍 **Mapa Google Maps** - Cobertura en Cartagena
♿ **Accesibilidad WCAG** - Cumple estándares web

## Tecnologías

- **HTML5** - Estructura semántica
- **CSS3** - Diseño responsive con Tailwind CSS
- **JavaScript Vanilla** - Interactividad sin dependencias
- **Google Maps API** - Integración de ubicación
- **WhatsApp API** - Sistema de contacto

## Servicios

- Aseo corporativo y oficinas
- Hoteles y rentas cortas
- Limpieza post-obra
- Cocinas y restaurantes

## Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/tu-usuario/proyecto-grupo-limpia.git
cd proyecto-grupo-limpia
```

2. Abrir en navegador:
```bash
# Opción 1: Abrir directamente
open index.html

# Opción 2: Con live server (VS Code)
# Instalar extensión Live Server y hacer clic en "Go Live"
```

## Desarrollo

El proyecto no requiere instalación de dependencias. Simplemente edita `index.html` y actualiza los cambios en tu navegador.

### Pasos para modificar

1. Edita el archivo `index.html`
2. Actualiza videos en `assets/imagenes/`
3. Cambia la configuración de Google Analytics en el `<head>`
4. Personaliza el número de WhatsApp
